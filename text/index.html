P47: 記号

<?php などの記号は半角で書く。全角だと期待通りに動かない。
特に全角空白は分かりにくいので注意する。
全角空白を強調表示する拡張機能 zenkaku(by mosapride)→https://marketplace.visualstudio.com/items?itemName=mosapride.zenkaku

4章

P65: 結合演算子 
            $name = $name . "花子"; // ←のように書いてもOK

=の右側の処理『$name と"花子"を繋げる』を実行して、その結果を$nameに入れる
通常は左から処理していくが、=が出てきたら注意する

P68: 四則演算と()

5章

P78: リスト5-1

日付を変えないと何も表示されない。 mktime(...)のところを今日の日付にする。
日本では、2023年4月20日、のように年月日の順が普通だが、英語圏だと、April 20th, 2023のように、月日年の順。
1分=60秒。1時間=3,600秒。1日=86,400秒。

P79: 比較演算子

                        ...
                    }
